# Askme
## _Ask any question _

Ask any question and get an answer.
## Features
- #Hashtag function
- Easy registration
- Upload avatars
- Rating of users

## Installation

Download the app.

```sh
Install Ruby, Rails and Postgresql
rails db:migrate
rails s
```

Run the application

```sh
http://127.0.0.1:3000/
```
